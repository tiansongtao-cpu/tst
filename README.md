# 招生网站模板

这是一个适合部署到 GitHub Pages 的静态招生网站模板，目录里没有构建步骤，直接上传即可。

## 本地文件

- `index.html`：页面结构
- `styles.css`：页面样式
- `script.js`：滚动显示动画

## 发布到 GitHub Pages

1. 在 GitHub 新建一个仓库，例如 `admissions-site`
2. 把这几个文件上传到仓库根目录
3. 进入 GitHub 仓库的 `Settings`
4. 打开 `Pages`
5. 在 `Build and deployment` 中选择：
   - `Source`: `Deploy from a branch`
   - `Branch`: `main`
   - `Folder`: `/ (root)`
6. 保存后等待几分钟，GitHub 会生成一个公开网址

## 上线前建议替换的内容

- 网站标题和项目名称
- 导师或团队介绍
- 招生对象与要求
- 时间安排
- 邮箱、微信、二维码
- 报名表链接
- 学校或团队相关图片
